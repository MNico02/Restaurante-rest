# 🏪 Servicio REST - Proveedor de Restaurante

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)
![Spring Boot](https://img.shields.io/badge/Spring_Boot-6DB33F?style=for-the-badge&logo=spring-boot&logoColor=white)
![Protocolo](https://img.shields.io/badge/REST-005C84?style=for-the-badge)

Este repositorio contiene un servicio web independiente basado en API RESTful que simula el sistema interno de un restaurante. 

Este proyecto forma parte del ecosistema **Ristorino** y actúa como un proveedor externo. La aplicación principal de Ristorino se conecta a este servicio para consultar en tiempo real el menú, las promociones y la disponibilidad de mesas.

## 🔗 Ecosistema Ristorino
* Ristorino Frontend (Angular)(https://github.com/MNico02/Ristorino-Angular)
* Ristorino Backend Principal(https://github.com/MNico02/Ristorino-rest)

* **Lenguaje:** Java 
* **Framework:** Spring Boot 
* **Protocolo de Comunicación:** REST (JSON)
* **Base de Datos:** MySQL
